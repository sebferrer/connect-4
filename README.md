# connect-4
The Connect Four game with a dataset generator, playable by humans or AI

<p>http://sebferrer.fr/connect4</p>

## Build
```
npm install
npm start
```
This project building requires Node ans Gulp.
Node 12 isn't compatible with Gulp 3 or 4, so Node 11 is recommended.
### How to have multiple Node versions installed
- For Linux: https://www.npmjs.com/package/n
- For Windows: https://github.com/coreybutler/nvm-windows
